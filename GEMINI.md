# 프로젝트 가이드라인 (GEMINI.md)

## 1. 시스템 환경
- OS: Windows 11 64bit
- Python: 3.10.11 64bit (venv64)
- Office: Microsoft 365 유료 버전


## 2. 협업 방식
- 코드 제안 시 에디터 내용을 **직접 수정(Inline Edit)**하는 방식을 우선한다.
- 모든 주석은 **한국어**로 작성한다.
- 사용자와 천천히 호흡을 맞추며 진행한다.
- 반복되는 오류로 사용자를 지키게 하기 보다는 깊게          기억, 능력의 한계가 오면 즉시 솔직히 이야기 한다.
